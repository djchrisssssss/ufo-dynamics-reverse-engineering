## 3. 驗證方法

### 第一層 — 來源選擇標準

1. **同儕審查文獻** — 刊載於公認期刊（Nature、Science、Physical Review、Classical and Quantum Gravity）
2. **政府文件** — ODNI、AARO、DoD 之正式出版物
3. **DIRD** — 由 DIA 在 AAWSAP 計畫下委託之專家技術報告
4. **教科書** — 標準參考書（Misner-Thorne-Wheeler、Wald、Visser）

### 第二層 — 四步驟資料點驗證

1. **來源識別** — 為每項統計數據識別主要權威來源
2. **交叉比對** — 與至少一個次要來源比較（記錄於 `verification_note`）
3. **情境合理性** — 與相關資料點之一致性檢查
4. **文件記錄** — 於 JSON 中記錄引用字串或來源物件、存取日期及驗證說明

### 第三層 — JSON Schema 強制驗證

所有統計檔案均依 `data/schemas/statistics-schema.json` 驗證：
- ID 格式：`^(s[0-9]{2}|sA[0-9])-[0-9]{3}$`
- 必填欄位：`id`、`metric`、`value`、`unit`、`date`、`source`（引用字串或結構化來源物件）
- 信心等級：`high`、`moderate`、`low`
- 驗證狀態：`verified`、`theoretical`、`speculative`

## 7. 已知限制

1. **自動化 HTTP 檢查並不完美** — 部分有效的官方或 DOI 端點會因機器人限制而對腳本請求返回 HTTP 403
2. **DIRD 內容驗證** — 個別 DIRD 技術主張未獨立驗證；視為專家意見
3. **NIF 數據時效性** — 核融合進展數據更新迅速。2025 年 4 月增益 4.13 數據已於 2026 年 3 月對照 LLNL 及 DOE 來源獨立驗證（src-059）。後續實驗可能突破此紀錄。
4. **USS Nimitz 加速度** — 衍生自雷達追蹤數據，其校準與詮釋在 UAP 研究社群中仍有爭議
5. **訂閱來源** — 部分 DOI 連結論文需期刊訂閱方能完整存取
6. **註冊表覆蓋範圍** — URL 註冊表屬抽查性質，不能保證所有引用來源在各地區或網路環境下皆可直接瀏覽
